Définitions de dimensions etc. pour faciliter la chose : 

 - taille des pièces : on choisit de définir des pièces 'tileable'. Une pièce peut prendre plusieurs 'tiles' de place, mais la taille des 'tiles' est fixée à 16 caractères de large (<->) par 8 de haut (^-v). Le modèle est pris avec le symbole '%' notamment pour faciliter le développement : subject to change.
 - doors might be defined as 0 1 2 3 rather that 'north', 'east', 'south' 'west'
